# VR Holiday Homes Website
This is the repository for the VR Holiday Homes website, offering premium stays & luxury villas in Candolim, Goa.

## Tech Stack
- React 19
- Vite 8
- Tailwind CSS v4

## Local Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
